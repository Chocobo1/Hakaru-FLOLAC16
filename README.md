# Hakaru-FLOLAC16

# How to run this


1. [Install stack](http://docs.haskellstack.org/en/stable/README/#how-to-install)
2. Clone this repo

  ```shell
  git clone git@github.com:banacorn/Hakaru-FLOLAC16.git
  ```

3. Build

  ```shell
  cd Hakaru-FLOLAC16
  stack setup
  stack build
  ```

4. Run
  
  ```
  stack exec hakaru-exe 10 10000
  ```
